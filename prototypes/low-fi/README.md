# Prototipos Low-Fidelity – ActaSeguro MVP

Este directorio contiene los bocetos de baja fidelidad del sistema **ActaSeguro**, diseñado para mejorar la transparencia y confianza en la captura y validación de actas electorales.

##  Flujos cubiertos
- Operador de Mesa: Captura offline + validación + adjuntar foto.
- Supervisor: Revisión de alertas, aprobación/rechazo con bitácora.
